Это карточный каталог, реализованный с помощью HTML, CSS и JavaScript, с функциями поиска и фильтрации элементов. В проекте реализован интерактивный поисковик и фильтры, позволяющие быстро находить нужные карточки по разным параметрам. Проект предназначен для практики создания динамических интерфейсов и работы с DOM с помощью JavaScript.

Стек технологий
HTML5, CSS3, ES6+ (JavaScript), Git, SCSS, подключенные шрифты

Как открыть проект

Скачайте архив с файлами или склонируйте репозиторий.

Распакуйте архив и откройте папку с проектом.

В папке найдите файл index.html.

Откройте файл index.html в любом современном браузере (например, Chrome, Firefox или Edge) двойным кликом или через меню "Открыть с помощью".
